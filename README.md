# Intune Admin Scripts

Welcome to the **Intune** repository!  
This repository contains a wide variety of scripts and resources for administrators managing Microsoft Intune and related environments. Most content is provided in PowerShell for Windows automation and management.

## Features & Directory Overview

- **Application Management:** Scripts and tools for automating the deployment and configuration of applications in Intune.
- **Azure:** Automation and helper scripts related to Azure integration.
- **Conditional Access Policies:** Templates and scripts for managing Conditional Access configurations.
- **Device Management:** Tools for registering, managing, and remediating devices within Intune.
- **ImageDeploy:** Solutions for OS image deployment and provisioning.
- **InstallApps:** Scripts for bulk app installation and management.
- **Intune Management:** Core utilities for Intune administration.
- **Microsoft Store Update backend / Ms Store:** Resources for managing Microsoft Store apps and updates.
- **OLd:** Legacy or deprecated scripts kept for reference.
- **SCCM:** Integration samples for co-managed environments using SCCM with Intune.
- **Scripting:** Miscellaneous PowerShell scripts to enhance automation.
- **TKM-Uninstaller-V.1 & TKM-WinAppInstaller:** Custom installers and uninstallers.
- **Tss Logs:** Logging utilities and sample log output.
- **User handling:** Tools for user provisioning and management.
- **Windows 10 ESU Activation-Intune Remediation:** Remediation scripts for ESU activation.
- **deletenow / pr3 / intune Scripts Validator:** Other specialized scripts.

## Getting Started

Each directory contains specific scripts with details in comments at the top. Most scripts are intended for use by IT administrators with appropriate privileges.

### Requirements

- Windows environment (PowerShell 5.1 or later recommended)
- Microsoft Intune administrative access
- (Optional) Azure and SCCM permissions for hybrid use-cases

### Usage

1. Clone the repo:  
   `git clone https://github.com/thekingsmakers/Intune.git`
2. Browse to the relevant directory.
3. Read the script header for usage instructions, prerequisites, and description.
4. Run or adapt scripts as needed in your environment.
